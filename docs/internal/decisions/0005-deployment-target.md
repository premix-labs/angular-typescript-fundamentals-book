# 0005 Delivery Scope

## Status

Accepted

## Decision

- teach Angular production build and local static-output verification
- explain SPA server fallback requirements conceptually
- verify mock API is not unintentionally enabled in the production bundle
- do not implement cloud hosting, CI/CD, domains, analytics or production monitoring

## Consequences

- learners distinguish the Angular development server from deployable output
- hosting configuration remains a dedicated follow-up
