# สร้าง Angular 22 Project แบบ Standalone, Zoneless และ Strict

> สถานะบท: โครงร่างสำหรับพัฒนาต้นฉบับ

## เป้าหมาย

- สร้าง `product-catalog-angular` ด้วย Angular CLI 22.0.6
- เปิด standalone components, routing, strict mode, zoneless change detection, Vitest และ 2025 file naming ตั้งแต่ต้น
- อ่านไฟล์สำคัญใน workspace โดยไม่พยายามจำทุก configuration
- รัน development server, tests และ production build

## Milestone

สร้าง progressive snapshot แรกที่แสดง Product Catalog shell และผ่านคำสั่งตรวจทั้งหมด

## Checkpoint

ผู้เรียนเปิดแอปจาก URL ที่ CLI แสดง รีเฟรชได้โดยไม่มี console error และอธิบายความต่างระหว่าง development server กับ production build ได้
