# เตรียม Node.js 24, npm, Angular CLI 22 และ DevTools

> สถานะบท: โครงร่างสำหรับพัฒนาต้นฉบับ

## เป้าหมาย

- ตรวจ Node.js 24 LTS และ npm ให้ตรงกับเวอร์ชันที่ Angular CLI 22 รองรับ
- เรียก Angular CLI 22.0.6 แบบทำซ้ำได้โดยไม่ต้องติดตั้ง global CLI
- เตรียม VS Code, Angular Language Service และ Angular DevTools
- อ่าน terminal, browser console และ network panel

## Milestone

ผ่าน environment checklist โดยยังไม่ติดตั้ง package แบบ global โดยไม่จำเป็น

## Checkpoint

ผู้เรียนแสดงผลเวอร์ชัน Node.js, npm และ Angular CLI พร้อมอธิบายว่าจะอ่าน runtime URL จากที่ใด
